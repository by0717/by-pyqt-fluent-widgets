# 🧪 集成功能快速测试指南

## 📋 测试前准备

1. **启动应用** 
   ```bash
   python -m gallery  # 或者运行相应的启动脚本
   ```

2. **应用启动后** - 你应该看到多个标签页

---

## 🧪 测试场景 1: 验证原有功能 (执行全局配置)

### 目标: 确保"执行全局配置"功能仍然正常

**步骤**:
1. 打开 **BasicInputInterface** 标签页
2. 在"全局配置"下拉框中选择一个配置
3. 点击任意窗口的 **"开始执行"** 按钮
4. **预期结果**:
   - 窗口进度条开始更新
   - 日志区域显示执行日志
   - 显示"测试进行中..."状态
   - 最后显示 PASS/FAIL 状态

✅ **测试通过** 如果按照预期执行

---

## 🧪 测试场景 2: 验证新功能 (执行测试项目)

### 目标: 测试 TestRunner → BasicInterface 的信号传递

**步骤**:

#### Step 1: 在 TestRunner 中选择测试
1. 切换到 **TestRunnerUI** 标签页
2. 点击 **"按钮1选择器"** 标签页
3. 在测试项目列表中勾选若干测试项目
4. 观察状态: 应该显示"已为按钮1选择X个测试项目"

#### Step 2: 发送测试到执行界面
1. **使用代码调用** (当前代码中没有按钮，需要代码调用)
   
   在 Python 控制台或代码中:
   ```python
   # 获取 TestRunnerUI 实例
   test_runner = main_window.materialInterface
   
   # 调用发送方法
   test_runner.send_tests_to_interface(1)  # 发送按钮1的测试
   ```

2. **或者** 添加按钮后点击按钮 (推荐)
   - 在 TestRunnerUI 中为每个选择器添加 "发送到执行界面" 按钮

#### Step 3: 在 BasicInterface 中查看执行结果
1. 切换到 **BasicInputInterface** 标签页
2. 观察 **"窗口1"** 
3. **预期结果**:
   - 窗口状态变为"执行中..."
   - 日志显示 "🚀 开始执行测试项目: 共 X 个测试"
   - 循环显示每个测试的执行情况
   - 最后显示 "🎉 所有测试执行完成"
   - 状态显示 PASS

✅ **测试通过** 如果测试项目成功执行

---

## 🧪 测试场景 3: 多窗口并行执行

### 目标: 验证多个窗口可以同时执行不同的任务

**步骤**:
1. 在 BasicInterface 中:
   - 窗口1: 选择配置，点击"开始执行"
   - 窗口2: 从 TestRunner 发送测试并执行
   - 窗口3: 点击"开始执行"配置

2. **预期结果**:
   - 3 个窗口同时执行，互不干扰
   - 每个窗口显示独立的进度条和日志
   - 完成时间不同

✅ **测试通过** 如果 3 个窗口可以并行执行

## 🐛 排查问题

### 问题 1: BasicInterface 没有接收到测试

**检查步骤**:
1. 打开浏览器开发者工具或控制台
2. 查看是否有错误日志
3. 确认 TestRunnerUI 发出了信号:
   ```
   ✅ 已向执行界面发送 X 个测试项目
   ```
4. 确认 ExecutionInterface 接收到了信号:
   ```
   ✅ 按钮 0 已接收 X 个测试项目
   ```

### 问题 2: 执行时出现异常

**检查步骤**:
1. 查看输出日志中的错误信息
2. 确认 test_manager 是否正确初始化
3. 确认测试ID是否有效

### 问题 3: 信号没有连接

**检查步骤**:
1. 在 MainWindow 的 connectSignalToSlot() 中添加调试:
   ```python
   print("Connecting signals...")
   self.materialInterface.tests_sent.connect(
       self.basicInputInterface.receive_tests_from_runner
   )
   print("Signals connected!")
   ```
2. 查看是否打印了连接消息

## 🚀 快速测试脚本

如果想快速测试，可以在 Python 控制台执行:

```python
# 1. 获取主窗口实例
main_window = ...  # 你的主窗口实例

# 2. 测试信号连接
print("Testing signal connection...")
test_runner = main_window.materialInterface
exec_interface = main_window.basicInputInterface

# 3. 模拟选择测试
test_runner.test_sets[1] = ['test_001', 'test_002']

# 4. 发送测试
print("Sending tests...")
test_runner.send_tests_to_interface(1)

# 5. 检查日志输出
# 应该看到 "✅ 已向执行界面发送 2 个测试项目"
# 应该看到 "✅ 按钮 0 已接收 2 个测试项目"
```
